## Installation

To install the CLI Tool on your machine follow these steps:

Make it executable permissions using 

```bash
  chmod +x qte.sh
```

Then, create a symlink to the script using

```bash
  sudo ln -s $(pwd)/qte.sh /usr/local/bin/qte
```
